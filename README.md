YourHealth+

YourHealth+ — це веб-платформа, яка слугує інтерактивною бібліотекою дієт та харчових планів, орієнтованих на користувачів із харчовими обмеженнями та особливими потребами.
Сервіс дозволяє легко підібрати раціон для випадків:  
• алергій та непереносимостей (глютен, лактоза, горіхи тощо);  
• захворювань шлунково-кишкового тракту (гастрит, виразкова хвороба, панкреатит);  
• необхідності спеціальних дієт (низькокалорійні, високобілкові, м’які дієти);  
• інших станів, де важливе індивідуальне харчування.  

Проєкт реалізує адаптивну верстку, сучасні підходи до стилізації, окремі CSS-модулі та кросбраузерність.  

Структура проєкту  
• css  
       base.css         — базові стилі (шрифти, заголовки, абзаци)  
       components.css   — стилізація кнопок, форм, меню  
       layout.css       — сітки, макет сторінок
       style.css        — головний файл, що імпортує всі стилі  

• images             — зображення та медіа (логотипи, фото, іконки)  
• js                 — На даному етапі папка пуста  

index.html          — головна сторінка  
diets.html          — каталог дієт  
recipes.html        — розділ рецептів  
communities.html    — сторінка спільнот  
profile.html        — профіль користувача  
about.html          — інформація про платформу  
contacts.html       — контакти
